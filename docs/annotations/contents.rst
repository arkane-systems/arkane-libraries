Annotations
===========

    namespace ArkaneSystems.Arkane.Annotations

This namespace, found in the *Arkane.Annotations* assembly/NuGet package, contains annotative attributes which add information to an assembly or its members, chiefly for 
the use of external tools, but which can also be conveniently accessed via reflection.
